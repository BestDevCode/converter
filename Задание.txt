1. Создать интерфейс IConverter, который будет содержать метод convert();
1.2 Метод convert() должен принимать в качестве входящих параметров исходную сумму, исходную валюту, выходную валюту. Результат метода будет 
иметь тип ConvertionResult.
1.3 Тип ConvertionResult должен содержать в качестве полей класса
        - код результата операции типа OperationResult (enum: SUCCESS, ERROR)
        - сообщение об ошибке (строка, может отсутствовать, если ошибки нет)
        - исходные данные (например inputCurrency, inputAmount)
        - результат (resultCurrency, resultAmount)
        - курс
2. Создать класс ковертер (Converter), который реализует  интерфейс IConverter
3. В конструктор класса Converter передается клиент (Client) и оператор (Opertor), которые создаются ранее (например в методе main)
4. В методе convert() мы должны проверить, можем ли мы обслужить этого клиента. Этим занимается operator, т.е. мы вызываем метод chek() у opertator.
Проверка заключается например в наличии документа у клиента
5. После того, как мы проверили клиента, мы начинаем конвертацию:
    - Узнаем текущий курс. Для этого у нас должен быть класс с методом  получения текущего курса. 
        Например CentralBankService и у него есть метод getCureentRate()
    - Operator создает заявку на конвертацию, значит в классе Operator должен быть метод createConvertionRequest(Client, CurrencyFrom, 
    AmountFrom, CurrencyTo, CurrentRate), и результатом выполнения метода будет заявка ConvertionRequest.
    - Далее ковертер достает из заявки необходимые данные и выполняет ковертацию. При необходимости можно добавить проверку
     перед конвертацией (например входная сумма больше 0)
    - Результатом конветnации будет являться переменная типа 
    ConvertionResult. Код результат операции присваивается в зависимости от успеха или неуспеха операции. Успех или неуспех может зависить
    как от вышестоящей проверки, так и от результа самой операции  конвертации. Сообщение об ошибке есть в случае кода 
    операции != SUCCESS. Исходные данные мы знаем. Результат мы получили.
    Курс мы тоже получиил ранее.
6. Создаем класс принтер, который может печатать ConvertionResult.
Например:

Операция выполнена: Успешно
Получено: 100$
Выдано: 6000р
Текущий курс: 1$ = 60р

Операция выполнена: Неуспешно
Сообщение об ошибке: Не удалось получить текущий курс.
